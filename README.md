# ListenHub

**Explain Anything. In Videos, Podcasts, and More.**

Make knowledge click. Bring stories to life.

---

You have ideas worth sharing. ListenHub turns them into content people actually want to watch and listen to — no editing skills required.

## Install

**Vercel skills.sh**
```bash
npx skills add marswaveai/orange-skill
```

**Open Agent Skills**
```bash
bunx add-skill marswaveai/orange-skill
```

**Claude Code Plugin**
```bash
claude plugin install marswaveai/orange-skill
```

## What You Can Create

| Say this... | Get this |
|-------------|----------|
| "Make a podcast about quantum computing" | Two hosts discussing your topic |
| "Explain this article as a video" | Narrated explainer with AI visuals |
| "Read this aloud" | Natural voice audio |
| "Generate an image: sunset over mountains" | AI-generated artwork |

Just describe what you want. ListenHub handles the rest.

## Supported Inputs

- Any topic you can describe
- YouTube videos
- Article URLs
- Plain text
- Image prompts
