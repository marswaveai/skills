You have ideas worth sharing. [ListenHub](https://listenhub.ai) turns them into content people actually want to watch and listen to — no editing skills required.

## Install

```bash
npx skills add marswaveai/skills
```

## What You Can Create

| Say this...                                | Get this                           |
| ------------------------------------------ | ---------------------------------- |
| "Make a podcast about quantum computing"   | Two hosts discussing your topic    |
| "Explain this article as a video"          | Narrated explainer with AI visuals |
| "Read this aloud"                          | Natural voice audio                |
| "Generate an image: sunset over mountains" | AI-generated artwork               |

Just describe what you want. ListenHub handles the rest.

## Supported Inputs

- Any topic you can describe
- YouTube videos
- Article URLs
- Plain text
- Image prompts

## Setup

**ListenHub API Key** — [Get yours](https://listenhub.ai/settings/api-keys) (Pro plan required)

**Labnana API Key** — [Get yours](https://labnana.com/api-keys) (Pro plan required)

Keys auto-configure on first use.

**CLI Output Language**: The Skills CLI automatically detects your system language and outputs in English or Chinese accordingly.

## Supported Clients

Claude Code · Cursor · Windsurf · OpenCode · Codex · Trae · and more.
